Mac下MySQL8.0版本源码编译调试

Posted on 2023-06-20 Edited on 2024-07-18 In mysql

下载源码

clion导入mysql项目

mac下采用clion idea 导入mysql_server项目，并进行cmake配置

其中比较重要参数有

  • DDOWNLOAD_BOOST=1 如若没有boost则进行下载，默认是没有的
  • DWITH_BOOST boost下载路径
  • DCMAKE_BUILD_TYPE 表示debug
  • 其余则为编译和数据目录

常见错误

openssl

正常mac下安装 brew install openssl 是安装v3以上版本，但是mysql8.0需要openssl1.1版本。

bison

缺少bison库
